# HW1_WEBDESIGN
homework for Web Design course
Chosen scenario: Local Small Business Digital Presence

Implemented requirements:
- index.html
- contact.html
- data tables
- style css
- <!doctype html>
- lang attribute
- meta viewport
- meta charset (UTF-8)
- nav bar for all pages
- data page with minimum 4 rows of data
- form page with minimum 5 controls total, including name, email, subject/topic, message, and submit button
- one external stylesheet used by all pages
- HTML and CSS validation


Known limitations: Form input handling can't be done without JavaScript
